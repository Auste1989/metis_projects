****************************************English Textual Entailment Graph DatasetV2.031 December 2014****************************************1. OverviewThis dataset was created within the EU-funded project EXCITEMENT - EXploring Customer Interactions through Textual EntailMENT (http://www.excitement-project.eu/), and is the result of a join effort of:- NICE SYSTEMS LTD, Israel- FONDAZIONE BRUNO KESSLER (FBK), Italy- BAR ILAN UNIVERSITY (BIU), Israel- DEUTSCHES FORSCHUNGSZENTRUM FUER KUENSTLICHE INTELLIGENZ GMBH (DFKI), GermanyThe dataset is freely distributed for research purposes under a Creative Commons license Attribution-NonCommercial-ShareAlike (http://creativecommons.org/licenses/by-nc-sa/4.0/). Publications or presentations containing research results obtained through the use of the Textual Entailment Graph dataset should acknowledge its creators.2. Contents./dataThis directory contains three datasets:1) MODIFIER_ANNOTATION_DATASET_ENG_V2.02) ENTAILMENT_GRAPH_DATASET_ENG_V2.03) ENTAILMENT_GRAPH_RTE_FORMAT_ENG_V2.0./docsThis directory contains:- This README file- Documentation about dataset purpose, data structure, and format (pdf)- Statistics for MODIFIER_ANNOTATION_DATASET_ENG_V2.0 (xlsx)- Statistics for ENTAILMENT_GRAPH_DATASET_ENG_V2.0 (xlsx)
- Documentation about the ENTAILMENT_GRAPH_RTE_FORMAT dataset (txt)3. Contact InformationFor further information about this data release, contact the following people:Luisa Bentivogli (FBK)  bentivo@fbk.euBernardo Magnini (FBK) magnini@fbk.euLili Kotlerman (BIU) lili.dav@gmail.com
